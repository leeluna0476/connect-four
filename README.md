# connect-four
WebSocket practice.

## env settings
1. Make a python virtual environment.
```bash
# Mac
$ python3 -m venv [venv_name]

# activate
$ source [venv_dir]/activate

# deactivate
$ deactivate
``` 
2. Install websockets library in the virtual environment.
```bash
([venv_name]) $ pip install websockets
```
